## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation
1. Clone or download the project files
2. Open `index.html` in your web browser
3. Start planning your 2026 goals!

### Usage
1. **Dashboard**: View your overall progress and statistics
2. **Goals**: Add and manage your personal goals
3. **Projects**: Organize goals into larger projects
4. **Notes**: Capture ideas and thoughts
5. **Rules**: Set personal guidelines and motivation
6. **Skills**: Track new skills and achievements

## Data Storage

All data is stored locally in your browser's LocalStorage. This means:
- Data persists between sessions
- No internet connection required after initial load
- Data is private and stored only on your device
- Clearing browser data will remove all stored information
